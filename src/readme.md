Vhello_rust/

├── src/             # Source code directory

### Key Concepts Covered
* **Project Structure:** Understanding the relationship between `Cargo.toml` and the `src/` directory.
* **Ownership & Borrowing:** Demonstrating how Rust manages memory without a garbage collector.
* **Pattern Matching:** Using the powerful `match` expression for control flow.
* **Interactive I/O:** Capturing and processing user input in the terminal.
