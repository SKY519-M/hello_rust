# Rust Beginner Toolkit

An interactive command-line toolkit built with Rust to demonstrate core programming concepts.  
**Moringa School AI Capstone Project**

---

## ✨ Features

- Interactive menu system
- Student score management
- Demonstrates key Rust concepts:
  - Structs & Methods
  - Ownership & Borrowing
  - Pattern Matching
  - Vectors (`Vec<T>`)
  - User Input Handling
  - Loops
